AI Brain Configuration

Switch Sprout from lookup mode to generative mode. When enabled, Sprout will think about your question, consult its knowledge base, personality, and directives, then craft an original response — like a real mind.

🔴 Mode: Lookup (keyword matching)

Use a proxy endpoint if calling from the browser (CORS). Direct Anthropic API works from server-side or CORS-enabled proxies.

How it works:
When generative mode is active, Sprout assembles its full "mind" — identity, personality traits, directives, writing style, and knowledge base — into a rich context. It then sends your question along with this context to an LLM, which generates a truly original response in Sprout's voice.

Lookup mode (fallback):
If the LLM is unavailable or not configured, Sprout falls back to keyword matching against training data — the classic behavior.
